# Cognitive Assessment Platform - Backend

This is the FastAPI backend service for the Cognitive Assessment Platform (CAP), providing APIs for clinical assessments, patient management, doctor dashboards, and PDF report generation.

## Technologies
- **Framework**: FastAPI (Python)
- **Database**: PostgreSQL (via SQLAlchemy asyncpg)
- **Migrations**: Alembic
- **PDF Generation**: ReportLab
- **Testing**: Pytest
